# Downloading data

Télécharger les données sur le Home page de M. Maignien.

``` bash
wget https://pagesperso.univ-brest.fr/~maignien/teaching/M1-MFA/UE-Ecogenomique2/EcoG2_data_cc2.tar.gz
```

# Decompress the data

``` bash
tar xvzf EcoG2_data_cc2.tar.gz
```
